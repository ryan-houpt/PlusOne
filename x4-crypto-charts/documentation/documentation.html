Installation

Step #1: Download plugin

Download the plugin main ZIP archive ("All files & documentation") from your CodeCanyon downloads page (https://codecanyon.net/downloads). It should look like this:

codecanyon-21534017-x4-crypto-charts-javascript-plugin.zip

Step #2: Upload to your server

Unzip the downloaded ZIP file on your computer. You will have x4-crypto-charts folder containing everything related to the plugin. You can copy the entire folder to your server, or you can copy only the minified version of the plugin. To have the minified version, unzip the x4-crypto-charts-2.1.1.zip archive. You will have a folder with the same name, but with fewer files.

Connect to your server via FTP and copy the x4-crypto-charts folder to your website, where you keep other javascript plugins, such as jquery, ckeditor, angular, react, vue etc.

Step #3: Make files accessible

Make sure that all static files in the x4-crypto-charts folder are accessible through the browser, configure your web server if needed. For example, the plugin entry point file x4-crypto-charts.js should be accessible at:

https://[your_website_domain_name]/[path_to_plugins_folder]/x4-crypto-charts/x4-crypto-charts.js

e.g.

https://example.com/plugins/x4-crypto-charts/x4-crypto-charts.js

Step #4: Load plugin

Put the following line of code inside the <head> tag of your html page to load the plugin entry point file (preferably before other styles and scripts, because it loads asynchronously):

<script src="[url_of_entry_point]" data-entry="x4-crypto-charts" data-vc-enabled async></script>

[url_of_entry_point] - url of the plugin entry point file x4-crypto-charts.js, see Step #3.

If you don't want the Visual Customizer to be loaded on the page, you can use the following code:

<script src="[url_of_entry_point]" data-entry="x4-crypto-charts" async></script>

Step #5: Insert widget

Put the following code snippet inside the <body> tag somewhere you want to insert the widget on your html page:

<div id="main-widget"></div><script>window.X4CryptoCharts=window.X4CryptoCharts||[];window.X4CryptoCharts.push({"type":"lineChart","selector":"#main-widget"});</script>

The widget will automatically fit the full width of the parent container. You can change this in the Visual Customizer in the Layout section.

Available widget types:

  • lineChart

Replace "type":"lineChart" by any other available widget type if needed.

Step #6: Customize widget

Open your website page in the browser. You should see the widget and the Visual Customizer open automatically.

Use the Visual Customizer to customize your widget. After completing the customization, click the "Save Changes" button at the top of the Visual Customizer. You will see instructions for replacing the code snippet to apply your changes.

To view additional options, click on the "Show additional options" link. Here you can change the options of the Visual Customizer, or change the widget tag selector (id/class).

Updating Plugin

Step #1: Download plugin

Download the plugin main ZIP archive from your CodeCanyon downloads page (https://codecanyon.net/downloads).

Step #2: Replace on your server

Unzip the downloaded ZIP file to have the x4-crypto-charts folder, entire or minified.

Connect to your server via FTP and replace the x4-crypto-charts folder on your website.

Step #3: Clear browser cache

The URL of the plugin static files will remain the same. Therefore, if you use any cache software (such as Cloudflare, or custom caching), you need to add some suffix to the plugin entry point file, e.g. ?ver=2.1.1

https://example.com/plugins/x4-crypto-charts/x4-crypto-charts.js?ver=2.1.1

<script src="https://example.com/plugins/x4-crypto-charts/x4-crypto-charts.js?ver=2.1.1" data-entry="x4-crypto-charts" data-vc-enabled async></script>

This way you will be sure that all website visitors will load new files instead of cached in the browser.

Build from Source Files

If you need to make changes to the source files, you need to follow these steps:

Step #1: Install Node.js

Install Node.js on your computer by following the instructions at https://nodejs.org

Step #2: Run commands

Open a command-line console and run the following commands:

// go to the source-files folder inside the main ZIP archive
cd x4-crypto-charts/source-files

// install project dependencies
npm install

// run build process
npm run build

Step #3: Replace on your server

You will see the /dist folder created inside the source-files folder, it contains the newly generated static files. Connect to your server via FTP and replace those static files on your website.

Frequently Asked Questions

How to make the widget fit the full width of the parent container?

You can do that using the Visual Customizer. Find the following option: Base Options -> Layout -> Set fixed width . Deselect this option and the widget will fit the full width of the parent container.

How to display the Visual Customizer only for the admin users?

You can restrict access to the Visual Customizer by adding the data-vc-* attributes for admin user only. Suppose we have the is_admin() PHP function, which returns true only if the current user is admin.

Here is an example of how this can be implemented:

<script <?php echo (is_admin() ? 'data-vc-enabled' : ''); ?> src="https://example.com/plugins/x4-crypto-charts/x4-crypto-charts.js" data-entry="x4-crypto-charts" async></script>

How to invert colors and move to the right the Visual Customizer initially?

You can invert colors by adding the data-vc-invert attribute. You can move to the right by adding the data-vc-right attribute.

Here is an example of how this can be implemented:

<script src="https://example.com/plugins/x4-crypto-charts/x4-crypto-charts.js data-entry="x4-crypto-charts" data-vc-enabled data-vc-invert data-vc-right async></script>
